# 用語集

## 概要

このドキュメントでは、プロジェクトで使用される専門用語や略語を定義します。

## 用語一覧

### A

#### ADR (Architecture Decision Record)

アーキテクチャ決定記録。プロジェクトにおける重要な技術的決定とその理由を記録したドキュメント。

### C

#### cub3D

42の課題プロジェクト。Wolfenstein 3D風のレイキャスティングエンジンを実装する。

### D

#### DDA (Digital Differential Analyzer)

レイキャスティングで使用されるアルゴリズム。光線がマップのグリッドを通過する際の交差点を効率的に計算する。

### F

#### FOV (Field of View)

視野角。プレイヤーが一度に見ることができる範囲の角度。

### M

#### MiniLibX

42が提供するシンプルなグラフィックスライブラリ。X Window Systemを抽象化し、ウィンドウ作成や描画を簡単に行える。

### R

#### Raycasting

レイキャスティング。視点から光線を飛ばし、最初に衝突するオブジェクトを検出することで3D風の描画を行う技術。

### T

#### Texture mapping

テクスチャマッピング。3Dオブジェクトの表面に2D画像を貼り付ける技術。

## 用語追加のガイドライン

新しい用語を追加する際は、以下の形式に従ってください：

```markdown
#### 用語名 (略称があれば)

用語の説明。必要に応じて使用例や関連用語への参照を追加する。
```

用語はアルファベット順（日本語の場合は五十音順）に配置してください。
